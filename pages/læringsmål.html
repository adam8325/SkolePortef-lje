Individuelle læringsmål

UI/UX

20 ECTS
  • Viden - den studerende har viden om
    • Fundamentale principper om UI/UX såsom box-model, visuelt hieraki og alignment
    • De forskellige best practices, der anvendes ude i virksomhederne samt opmærksom på anti-patterns
    • Brugercentreret design, ved at forstå og analysere slutbrugerens arbejdsgang og ønsker samt krav
  • Færdigheder - den studerende kan
    • Anvende moderne design-værktøjer såsom Figma for at implementere og videreudvikle på layouts og wireframes
    • Udvikle og aligne simple layouts vha. moderne principper såsom flexbox og grid.
    • Opsætte og implementere moderne frontend-projekter, der følger best practice indenfor semantisk HTML og accessibility
    • Udvikle intuitive og interaktive projekter via et samspil mellem HTML, CSS og Javascript
    • Implementere responsive layouts, som sikrer den gode brugeroplevelse på tværs af forskellige enheder
  • Kompetencer - den studerende kan
    • Vurderer på tilvalg og fravalg af UI-elementer og valg af design
    • Hensigtsmæssigt tilpasse frontendens struktur, så den fremstår læsbar og uden code smells
    • Samarbejde og diskutere UI/UX løsninger med udviklere og undervisere for at sikre korrekt implementering

React.js

10 ECTS
  • Viden - den studerende har viden om
    • Komponenter og props, samt deres rolle i at implementere genanvendelige komponenter.
    • JSX: Lære, hvad JSX (JavaScript XML) er, og hvordan det bruges til at skrive HTML-lignende syntaks direkte i JavaScript-koden i React.
    • State-håndtering herunder hooks som UseState og UseEffect og integration af tredjepartsbiblioteker.
  • Færdigheder - den studerende kan
    • Bygge enkle, statiske og dynamiske webapplikationer med React.js ved at kombinere JavaScript og React
    • Anvende state med hooks: Lære at bruge React hooks som useState og useEffect til at håndtere state og sideeffekter i funktionelle komponenter.
    • Strukturere og opsætte et React-projekt baseret på brug af komponenter, props og hooks
  • Kompetencer - den studerende kan
    • Forklare og diskutere fordelene ved brug af React som moderne frontend framework
    • Perspektivere og sammenligne brugen og relevansen af React med andre frontend frameworks
    • Vurderer behovet for implementeringen af React i projektet iht. skalerbarhed og vedligeholdelse
    • Tilpasse eksisterende React-kodebase ved at samarbejde med andre uviklere og undervisere
